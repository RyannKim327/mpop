Home  We call this project as newbie designer, as still we are newbie. This design is inspired with Bootstrap's Panel. As also a practice to use gradients design, we decided to create a simple design that can be used by the newbies like us. Just enjoy using this, and read the following as basis.

Class Names

  • mpop-base
    • - As base of the panel. Tag base <div>
  • mpop-head
    • - Head or headings of the panel where you can add title and subtitles and tabs. Tag base <div>
  • mpop-body
    • - This is the body of the panel, where the contents are here. Tag base <div>
  • mpop-foot
    • - This is the footer of the panel. Tag base <div>
  • mpop-input
    • - This is where you can add input, like search boxes and logins. Tag base <span>
    - Use <mpop/> as line separator
  • mpop-tabs
    • - Tabs which is in listing method or using unordered list and list <ul> and <li>. Tag base <div>
  • mpop-field
    • - This is designed for fieldset, which is the legend is automatically modified. Tag base <fieldset>
  • mpop-list
    • - Designed for non-tabs listing. Tag based <ul> or <ol>
